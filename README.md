# Netflix Clone

[Demo](https://wenyi420.github.io/movies/)

## 使用技術
- Vue3 + Composition API 建置 SPA 頁面
- Vite 打包專案
- Vue Router 控管路由頁面
- Pinia 控管頁面資料、狀態
- 串接 GoogleSheet API 模擬後端資料庫
- 串接 FB 登入，搭配 GoogleSheet API  存取 FB 姓名與信箱作為帳戶資料。
- 串接 TMDB API，模擬 Netflix 電影資料呈現
- 使用 Ant Design Vue 作為 UI 框架

## 主要功能

- Dark / Light 網頁樣式切換
- 可註冊與登入帳戶或是用 FB 登入
- 可加入 / 移除我的片單功能
- 可透過搜索查找電影項目
- 可查看電影詳細資料

## 預計添加功能
